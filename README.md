## my dotfiles
though every application has a pretty minimal config, don't expect it to work on your machine without some tinkering.

### features
wm: yabai, sway (linux)

shell: zsh

prompt: ohmyposh

editor: nvim

terminal: ghostty (primary), wezterm

top bar: sketchybar

keybindings: skhd

colorscheme: kanagawa

font: 0xproto
